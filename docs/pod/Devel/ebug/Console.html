NAME

Devel::ebug::Console

VERSION

version 0.64

SYNOPSIS

  # it's easier to use the 'ebug' script
  use Devel::ebug::Console;
  my $console = Devel::ebug::Console->new();
  $console->run();

DESCRIPTION

Devel::ebug::Console is an interactive command-line front end to Devel::ebug. It is a simple Perl debugger, much like perl5db.pl.
