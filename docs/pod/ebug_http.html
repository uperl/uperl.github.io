NAME

ebug_http - A simple, extensible web Perl debugger

VERSION

version 0.36

SYNOPSIS

  % ebug_http calc.pl

DESCRIPTION

ebug is a web-based front end to Devel::ebug. It is a simple Perl debugger. When run, it will report back how to contact its front page.

Devel::ebug
