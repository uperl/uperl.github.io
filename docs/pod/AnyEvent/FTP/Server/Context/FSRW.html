NAME

AnyEvent::FTP::Server::Context::FSRW - FTP Server client context class with full read/write access

VERSION

version 0.19

SYNOPSIS

 use AnyEvent::FTP::Server;
 
 my $server = AnyEvent::FTP::Server->new(
   default_context => 'AnyEvent::FTP::Server::Context::FSRW',
 );

DESCRIPTION

This class provides a context for AnyEvent::FTP::Server which uses the actual filesystem to provide storage.

SUPER CLASS

This class inherits from

AnyEvent::FTP::Server::Context::FS

ROLES

This class consumes these roles:

AnyEvent::FTP::Server::Role::TransferPrep

COMMANDS

In addition to the commands provided by the above roles, and super class, this context provides these FTP commands:

RETR
NLST
LIST
STOR
APPE
STOU
